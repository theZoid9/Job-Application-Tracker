# Job-Application-Tracker

A web app where users can:

Add job applications
Track status (Applied, Interview, Rejected, Offer)
Add notes for each job
Filter/search applications
See stats (how many applied, interviews, etc.)


Core Features (MVP)

Start simple:

1. Add Job Application
Company name
Position
Status (dropdown)
Date applied
2. Display List
Show all jobs in a clean UI
3. Update Status
Change status with a dropdown
4. Delete Job
Remove application

🧨 Advanced Features (portfolio-level)
8. Backend (later)
Use Node.js + PostgreSQL (you’ve already touched this 🔥)
Save jobs in a database
9. Authentication
Login/signup system
10. Drag & Drop Board (like Trello)
Move jobs between columns
